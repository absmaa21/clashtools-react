# ClashTools
## Start the server
- Open a terminal
- Go into the directory
- `npm install`
- `npm run dev`

## How to use demo data (without java server)
- Open [env.ts](./src/env.ts) and change **skipNetwork** to **true** [](/src/env.ts)
